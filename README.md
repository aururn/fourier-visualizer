# 🌀 画像フーリエ変換 ビジュアル学習アプリ

画像の2次元離散フーリエ変換(2D DFT)を、数式・変換過程・結果のすべてをグラフィカルに見ながら学べる単一HTMLアプリです。ビルド不要・サーバー不要で、`index.html` をブラウザで開くだけで動きます(計算はすべてブラウザ内のJavaScript)。

## 機能

| セクション | 内容 |
|---|---|
| ① 数式 | 2D DFT・逆変換・振幅/位相スペクトルの定義(KaTeXで描画) |
| ② 入力画像 | 縞模様・市松・円盤・ガウシアンなど9種のサンプル+画像アップロード。各サンプルに「なぜこのスペクトルになるか」の解説付き |
| ③ 変換の過程 | 分離可能性を利用した「行FFT → 列FFT → fftshift」を1行・1列ずつアニメーション再生 |
| ④ スペクトル探索 | スペクトルをクリックすると対応する2次元正弦波(基底)と、実数値入りの数式 F(u,v) を表示。振幅のみ/位相のみ再構成の実験も |
| ⑤ フィルタ実験 | ローパス/ハイパス/バンドパス。「スペクトル × マスク → 逆FFT → 結果」のパイプラインをリアルタイム表示 |
| ⑥ 逐次再構成 | 振幅の大きい順に成分を足して画像が復元されていくアニメーション(復元エネルギー%表示) |

## 使い方

```
ブラウザで index.html を開くだけ
```

数式表示にKaTeX(CDN)を使用しているため、オフラインでは数式がプレーンテキストになります(動作自体は問題ありません)。

## 実装

- 128×128 グレースケール、基数2の反復型FFT(行→列の2段階)
- 依存ライブラリなし(KaTeXは表示のみ)
- 往復変換誤差 ~1e-15、正弦波スペクトルのピーク位置を検証済み
